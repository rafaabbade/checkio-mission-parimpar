Como posso saber se um número é par?

Existem algumas maneiras de realizar essa tarefa, uma delas é utilizando o operador matemático %. O % representa o resto de uma divisão.

Exemplos: 4%2 -> 0 5%2 -> 1 7%4 -> 3 100%3 -> 1

Ok, mas como posso usar isso pra descobrir se é par?

Podemos comparar o resultado dessa operação com outro valor. Se buscarmos o resto da divisão de um número par por 2, encontraremos o valor 0.

Para isso, podemos usar a estrutura de comparação a+b == c. Substitua com os termos corretos para obter uma resposta booleana.

O que é uma resposta booleana?

Os valores booleanos True e False são usados em Python para representar o resultado de expressões booleanas, que são expressões que podem ser avaliadas como True ou False. Os valores booleanos são muito úteis para controlar o fluxo de execução de um programa, usando estruturas condicionais como if, elif e else.
